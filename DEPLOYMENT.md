# 배포 체크리스트

## Render

1. GitHub 저장소를 Render에 연결한다.
2. `render.yaml` 기준으로 Web Service를 만든다.
3. 환경 변수를 넣는다.
   - `NAVER_CLIENT_ID`
   - `NAVER_CLIENT_SECRET`
4. 배포가 끝나면 Render가 준 HTTPS 주소를 확인한다.

## 아이폰 확인

1. 아이폰 Safari에서 Render 주소를 연다.
2. 화면이 잘 보이는지, 가로 스크롤이 없는지 확인한다.
3. 공유 버튼에서 `홈 화면에 추가`를 누른다.
4. 아이콘처럼 설치된 앱으로 다시 열어 본다.

## 점검 항목

- 홈 화면이 먼저 보이는가
- `현재 베스트셀러` 버튼이 따로 열리는가
- 내 서재 목록이 과밀하지 않은가
- 책 추가, 상세 보기, JSON 백업/복원이 되는가
- iPhone에서 좌우 흔들림이 없는가
