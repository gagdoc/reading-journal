# 나의 독서 일기

책 제목이나 ISBN으로 도서 정보를 불러와 완독일, 평점, 감상, 태그를 남기는 로컬 우선 웹앱입니다. 현재는 Naver Book Search를 사용하고, 기록은 브라우저 `localStorage`에 저장됩니다.

## 구조

- `index.html` - 앱 화면과 모바일 메타 설정
- `styles.css` - 서재/책장 느낌의 반응형 UI
- `src/app.js` - 화면 상태, 검색, 저장, 삭제 흐름
- `src/services/` - 책 검색 API 연동
- `src/state/` - 독서 기록 상태와 통계
- `src/ui/` - 카드 렌더링
- `server.js` - 정적 파일 서빙과 Naver API 프록시

## 실행

1. `.env` 파일에 Naver API 키를 넣습니다.
2. 개발 서버를 실행합니다.

```bash
npm run dev
```

브라우저에서 `http://localhost:8000/`을 열면 됩니다.

## iPhone에서 확인하기

1. PC와 iPhone을 같은 Wi-Fi에 연결합니다.
2. 서버 실행 후 터미널에 표시되는 `http://내-IP:8000/` 주소를 iPhone Safari에서 엽니다.
3. Safari 공유 메뉴에서 `홈 화면에 추가`를 선택하면 설치형 웹앱처럼 사용할 수 있습니다.

## 동작

- 첫 화면에는 읽은 책 목록이 보입니다.
- `책 추가`를 누르면 검색창과 기록 폼이 열립니다.
- 제목 일부 또는 ISBN으로 책을 찾고 선택한 뒤 저장합니다.
- 기록은 새로고침 후에도 유지됩니다.

## 참고

- 아이콘과 설치 메타를 넣어 iPhone/PWA 환경을 고려했습니다.
- 네트워크가 없으면 책 검색은 실패할 수 있지만, 저장된 기록은 그대로 남습니다.
